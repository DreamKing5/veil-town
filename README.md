# VEIL TOWN

招待制・4〜12人用のブラウザ型ソーシャル推理ゲームです。スマホ、PC、iPadから各プレイヤーが自分の端末で参加し、役職配布・昼夜進行・能力処理・投票・勝敗判定を自動で行います。

## 完全無料構成

初版は、以前の「イラストリンク」と同じ考え方を採用しています。

- **公開**: GitHub Pages または Vercel Hobby（静的HTML）
- **リアルタイム通信**: PeerJS + WebRTC
- **専用サーバー**: 不要
- **データベース**: 不要
- **月額費用**: 0円

参加者のうち部屋を作った端末が、プレイヤーを兼ねながら自動進行の計算を担当します。専用ゲームマスター端末は不要です。ゲーム中は作成者のタブを閉じないでください。

## 公開方法（GitHub Pages）

1. GitHubで新しいリポジトリを作る。
2. このフォルダの `index.html` をアップロードする。
3. `Settings → Pages → Deploy from a branch` を選ぶ。
4. `main / root` を指定して保存する。
5. 表示されたURLを開き、「新しい部屋を作る」を押す。
6. ゲーム画面の「招待URLをコピー」を押し、参加者だけに送る。

## 公開方法（Vercel）

1. Vercelで `Add New → Project` を開く。
2. このフォルダを含むGitHubリポジトリを選ぶ。
3. Framework Presetは `Other`、Build Commandは空欄のまま公開する。
4. 発行されたURLから部屋を作成する。

個人・非商用の仲間内利用を想定しています。各サービスの無料枠と利用条件は変更される場合があります。

## 操作

1. 部屋作成者がプレイヤー名を入力して部屋を作る。
2. 招待URLを参加者へ共有する。
3. 参加者は名前を入力し、準備OKにする。
4. 4〜12人がそろったら作成者が開始する。
5. 各端末に秘密の役職が表示される。
6. 町長選挙 → 夜の能力 → 夜明け → 議論 → 追放投票を繰り返す。

反逆陣営は夜に「反逆陣営の襲撃先を選ぶ」ボタンを押し、襲撃モードへ切り替えて対象を選択します。スマホ・タブレットでも同じ操作です。

## 収録役職

医師、錯乱者、警官、見張り、調査員、罠師、密告者、挑発者、追跡者、清掃屋、濡れ衣師、生存者、連続殺人犯、爆弾魔、怪盗、亡霊、魔術師。

## 注意点

- ゲーム状態は部屋作成者のブラウザメモリにあります。作成者がページを閉じると、そのゲームは終了します。
- PeerJSの接続仲介とCDNを利用するため、インターネット接続が必要です。
- 招待キーは推測されにくい14文字ですが、URLを転送された人は参加できます。
- 本家の画像、音声、ロゴ、文章、画面素材は含んでいません。ゲーム進行を参考にした独自UI実装です。
- 本番利用前に、実際に使うiPhone Safari、Android Chrome、iPad Safari、PC Chromeで接続確認してください。
